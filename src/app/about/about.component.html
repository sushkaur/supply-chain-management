About This Application

Welcome to the Supply Chain Management System – a streamlined web application designed to help businesses efficiently track, manage, and update inventory and orders.

This application allows Admins, Suppliers, and Buyers to perform key supply chain operations like:

  • Managing inventory items with real-time updates
  • Tracking orders and shipment progress
  • Analyzing sales and user growth with visual charts
  • Secure login and role-based access for better control

Built with Angular and Node.js, the system ensures a responsive UI, fast backend API, and role-specific access, making it a powerful tool for modern supply chain operations.
